# Employee Management System - Task 02

### Features
- Full CRUD operations for employees: Create, Read, Update, Delete
- JWT Authentication with secure login/logout
- Password hashing using bcryptjs
- Server-side validation with express-validator
- SQLite database for data persistence
- Responsive UI with modern styling

### Tech Stack
**Backend:** Node.js, Express.js, SQLite, JWT, bcryptjs, express-validator 
**Frontend:** HTML5, CSS3, Vanilla JavaScript
